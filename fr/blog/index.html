Une conversation, une session CANFAR : Verbinal pour macOS 1.2.5

Verbinal pour macOS établit désormais un pont vers votre propre Claude Desktop ou Claude Code via le Model Context Protocol — ce qui vous permet de mener toute une session de recherche CANFAR au fil d’une seule pensée, en approuvant les actions qui comptent. Un survol de la nouvelle intégration de l’assistant IA, du modèle de contrôle qui la sous-tend et d’un flux de recherche de bout en bout.
