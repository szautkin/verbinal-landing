Windows 1.3.0 · Release · 6 July 2026

The whole research loop, native on Windows: Verbinal for Windows 1.3.0

Verbinal comes to Windows as a native WinUI 3 desktop for CANFAR and the CADC. Version 1.3.0 adds check-off Workflows, an AI assistant that can drive the app, a Direct3D 3D Cube Viewer, and a native Jupyter notebook with a local kernel — search to publication figure, without leaving the desktop.

macOS 1.3.0 · Release · 15 June 2026

From cube on disk to figure in the paper: Verbinal for macOS 1.3.0

Verbinal for macOS 1.3.0 adds the Cube Viewer — a native, GPU-accelerated 3D viewer for FITS spectral cubes. Orbit a cube in Volume mode, measure it with WCS and click-to-probe spectra in Slice mode, and export a publication-ready figure — all in one native window on your Mac.

macOS 1.2.5 · Release · 7 June 2026

One conversation, one CANFAR session: Verbinal for macOS 1.2.5

Verbinal for macOS now bridges to your own Claude Desktop or Claude Code over the Model Context Protocol — so you can run a whole CANFAR research session from a single train of thought, approving the moves that matter.
